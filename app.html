Android & iPhone

YUVA Club App

Install YUVA Club on your phone and use it as the official hub for registration, challenges, presentations, Zoom sessions, topic research, scoring, leadership profiles, points, tokens, certificates, rewards, and parent progress tracking.

Open Student Portal Register Student
Challenge Dashboard

Students can see program level, challenge stage, rubric score, sessions, topics, Zoom information, points, tokens, badges, and certificates.

Leadership Profile

The profile grows over time with presentations, speaking minutes, topics, feedback, milestones, and service hours.

Parent Dashboard

Parents can view progress, presentations, attendance, rewards, feedback, certificates, and recordings.

Admin Controls

Admins can approve students, assign sessions, manage topics, create challenge stages, score rubric categories, add points, tokens, awards, feedback, and certificates.

Safe by Design

No private student chat, parent contact connected to each account, admin approval, adult moderation, and a report issue button.

Global Topic Library

Students can choose topics in leadership, science, culture, business, history, health, sports, digital skills, STEM, and careers.

Same Library as Website

Full App Topic Library

The app uses the same topic library as the Yuva Club website. Students can open any topic page, prepare presentations, and submit their selected topic from the Student Portal.

Install on Android

Open yuvaclub.karmabro.com in Chrome, tap the menu, then choose Install App or Add to Home screen. The app icon will appear on the phone like a normal app.

Install on iPhone or iPad

Open yuvaclub.karmabro.com in Safari, tap Share, then choose Add to Home Screen. Use the YUVA Club icon to open the app.
