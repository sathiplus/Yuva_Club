Core Benefits

Built for Students, Parents, and Youth Organizations

YUVA Club is a secure installable web app. It works today in mobile and desktop browsers while native App Store and Google Play apps remain a future option.

01Guided Presentation Challenges

Students move from topic selection to research, practice, presentation, and feedback.

 02AI Coach and Rubric Feedback

AI-assisted feedback supports practice while official scoring remains reviewed by adults.

 03Leadership Progress

Students can track levels, activity, presentations, and next steps over time.

 04Certificates and Rewards

Certificates, points, tokens, and rewards are tied to approved student progress.

 05Digital Portfolio

Students build a record of presentations, leadership milestones, certificates, and service.

 06Parent Progress Tracking

Parents can access connected student information through the parent portal.

Install Guide

Install YUVA Club

Add YUVA Club to your home screen for faster access and an app-like experience.

Install directly from this browser

When supported, your browser can install YUVA Club with one tap.

1Open in Safari

Visit yuvaclub.app using Safari.

2Tap Share

Use the Safari share button at the bottom of the screen.

3Add to Home Screen

Select Add to Home Screen, then tap Add.

1Open in Chrome

Visit yuvaclub.app using Chrome or Edge.

2Open Browser Menu

Tap the three-dot menu.

3Install App

Choose Install app or Add to Home screen.

1Open Edge or Chrome

Visit yuvaclub.app on Windows.

2Look for Install

Use the install icon in the address bar when available.

3Pin for Access

Open YUVA Club from Start or the taskbar.

1Open Safari or Chrome

Visit yuvaclub.app on macOS.

2Add to Dock

Use the browser's app install option when supported.

3Launch Quickly

Return from the Dock or Applications.
